# Batch Web SDK

The Batch Web SDK allows you to build a meaningful communication experience in your website through highly personalized push notifications.

Our [ðŸ“• setup documentation](https://doc.batch.com/web/prerequisites) details the steps to take for an easy and successful integration.

# Prerequisites

Batch is compatible with the following browsers:

- Firefox 53 or higher
- Chrome 55 or higher
- Edge 75 or higher
- Safari 12 or higher
